1) Insertar cuenta. Solicitaremos todos los datos necesarios por scanner, crearemos el objeto cuenta y lo insertaremos en un array misCuentas.
2) Mostrar cuentas. Recorreremos el array misCuentas y mostraremos todas las cuentas bien formateadas.
3) Buscar. Solicitar el nombre de un titular y mostrar su saldo.
4) Filtrar. Mostrar las cuentas cuyo saldo sea mayor de 5000.
5) Salir.
