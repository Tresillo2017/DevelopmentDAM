

Se requiere crear un programa de gestión de una sala de ordenadores.

Cada puesto tiene las siguientes características: código de pc de tipo String, reservado de tipo booleano y persona de tipo persona que es por quien está reservado.

Una persona tiene de atributos DNI y nombre.

Crear clases, constructores, métodos get/set y toString .

Declarar un array listaOrdenadores que contenga objetos de tipo Puesto. Declararla en la clase Puesto.

Mostrar un menú que haga lo siguiente:

    Crear un puesto.
    Eliminar un puesto.
    Reservar un puesto.
    Listar los ordenadores reservados.
    Listar los ordenadores disponibles.
    Salir.
