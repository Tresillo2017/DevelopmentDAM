Una vez hecha la estructura de la tarea 2, crear un menú:

1) Crear Vehículos (deberá solicitar los datos del vehículo que se quiera crear, tendréis que saber si el vehículo es un barco, coche, avión... y almacernarlo en la lista misVehiculos.

2) Mostrar Información de un vehículo concreto.

3) Mostrar todos los coches (mostrar su número de serie, color y nº de puertas)

4) Solicitar un color y mostrar todos los vehículos de un color concreto.

5) Solicitar un tipo ( coche, barco o avión) y dependiendo del tipo mostrar todos los vehículos de ese tipo.

6) Salir.

AYUDA:

Usar instanceof para saber el tipo de dato del objeto
( objeto instanceof Coche --> esto retornaría true si el objeto es de tipo Coche)

Si necesitamos convertir un Vehiculo en Coche --> Haremos un casting  ((Coche) v) donde v es un objeto de tipo Vehiculo.
